Présentation du Logiciel

Ce projet est un autoclicker optimisé spécialement pour AZ Launcher, conçu pour offrir des performances stables, rapides et indétectables grâce à un timing software calibré. Aucune injection mémoire n’est utilisée, ce qui évite la détection instantanée et permet une compatibilité totale avec la protection anti-injection d’AZ Launcher.

La configuration inclut :
• Anti-Knockback : réduction à ~85 %
• Reach Multiplier : ~1.08x
• Base CPS : ~12 optimisé AZ
• Modes : Normal, No-Movement, Combo/W-TAP

Les contrôles permettent de changer de mode, activer/désactiver le clicker, ajuster KB/Reach/CPS, réinitialiser les statistiques ou quitter rapidement.

Fonctionne sans droits administrateur, totalement compatible AZ Launcher, discret et performant.
